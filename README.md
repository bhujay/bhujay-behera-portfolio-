## 🚀 How to Put on GitHub Pages (Free Hosting)

1. Go to [github.com](https://github.com) and sign in
2. Click **New Repository** → name it `bhujay-portfolio`
3. Set it to **Public**
4. Upload `index.html` and `style.css`
5. Go to **Settings → Pages → Source → main branch**
6. Your site will be live at `https://yourusername.github.io/bhujay-portfolio/`
